O que é um vetor?

Um vetor (ou array) é uma estrutura de dados que armazena uma sequência de valores do mesmo tipo, em posições contíguas de memória. Cada posição é identificada por um índice, que começa sempre em 0.

Imagine um bloco de gavetas numeradas: você não pode pular uma gaveta, e sabe exatamente em qual delas cada item está guardado.

Qualquer posição é acessada diretamente, sem varrer o vetor.
v[42]; // imediato
02

Declaração e inicialização

Em Java há três formas principais de criar um vetor:

Java
// Forma 1 — declarar e alocar depois
int[] idades;                      // declara a referência
idades = new int[5];               // aloca 5 posições (valores padrão: 0)

// Forma 2 — declarar e alocar na mesma linha
double[] notas = new double[4];    // valores padrão: 0.0
boolean[] flags = new boolean[3]; // valores padrão: false
String[] nomes = new String[2];   // valores padrão: null

// Forma 3 — inicialização literal (tamanho inferido)
int[] pares = { 2, 4, 6, 8, 10 };
String[] dias = { "Seg", "Ter", "Qua", "Qui", "Sex" };

// Obtendo o tamanho
System.out.println(pares.length);  // 5  (propriedade, não método!)
⚠️
Atenção: length é uma propriedade, não um método. Escreva v.length, nunca v.length() — isso causa erro de compilação.

03

Acesso e modificação de elementos

Cada elemento é acessado pelo índice entre colchetes. O índice válido vai de 0 até length - 1. Tentar acessar fora desse intervalo lança uma ArrayIndexOutOfBoundsException.

Java
int[] v = { 10, 20, 30, 40, 50 };
//          [0]  [1]  [2]  [3]  [4]

// Leitura
System.out.println(v[0]);   // 10
System.out.println(v[4]);   // 50  (último elemento)

// Modificação
v[2] = 99;
System.out.println(v[2]);   // 99

// ❌ Erro — índice fora do intervalo
System.out.println(v[5]);   // ArrayIndexOutOfBoundsException!

// ✅ Índice do último elemento: v.length - 1
System.out.println(v[v.length - 1]);  // 50
💡
Regra de ouro: índice válido é sempre 0 ≤ i < v.length. Guarde isso — é a causa número 1 de erros em vetores.
04

Percorrendo com for

O laço for e vetores foram feitos um para o outro. Há duas variantes: o for indexado (quando você precisa do índice) e o for-each (quando só precisa do valor).

Java — for indexado
int[] notas = { 7, 8, 9, 6, 10 };

// Leitura com índice
for (int i = 0; i < notas.length; i++) {
    System.out.println("notas[" + i + "] = " + notas[i]);
}

// Modificação — só possível com for indexado
for (int i = 0; i < notas.length; i++) {
    notas[i] = notas[i] + 1;   // bônus de 1 ponto
}

// Percorrer de trás para frente
for (int i = notas.length - 1; i >= 0; i--) {
    System.out.print(notas[i] + " ");   // 11 10 10 9 8
}
Java — for-each (enhanced for)
String[] frutas = { "Maçã", "Banana", "Laranja" };

// For-each: sintaxe limpa quando não precisa do índice
for (String fruta : frutas) {
    System.out.println(fruta);
}

// ⚠ For-each NÃO modifica o vetor original
for (int n : notas) {
    n = n + 1;   // modifica cópia local, não o vetor!
}

05

Operações clássicas

Java — soma, média, maior, menor
int[] v = { 4, 7, 2, 9, 5 };

// Soma
int soma = 0;
for (int n : v) soma += n;
System.out.println("Soma: " + soma);              // 27

// Média
double media = (double) soma / v.length;
System.out.println("Média: " + media);           // 5.4

// Maior elemento
int maior = v[0];
for (int i = 1; i < v.length; i++) {
    if (v[i] > maior) maior = v[i];
}
System.out.println("Maior: " + maior);           // 9

// Menor elemento
int menor = v[0];
for (int i = 1; i < v.length; i++) {
    if (v[i] < menor) menor = v[i];
}
System.out.println("Menor: " + menor);           // 2

// Busca linear (retorna índice ou -1)
int alvo = 9, posicao = -1;
for (int i = 0; i < v.length; i++) {
    if (v[i] == alvo) { posicao = i; break; }
}
System.out.println("Posição do " + alvo + ": " + posicao); // 3

// Contar elementos que satisfazem condição
int pares = 0;
for (int n : v) if (n % 2 == 0) pares++;
System.out.println("Pares: " + pares);           // 1
Java — copiar e inverter vetor
int[] original = { 1, 2, 3, 4, 5 };

// ❌ ERRADO — ambos apontam para o mesmo vetor
int[] errado = original;
errado[0] = 99;  // original[0] também vira 99!

// ✅ CORRETO — copiar elemento por elemento
int[] copia = new int[original.length];
for (int i = 0; i < original.length; i++) {
    copia[i] = original[i];
}

// Inverter vetor
int[] invertido = new int[original.length];
for (int i = 0; i < original.length; i++) {
    invertido[i] = original[original.length - 1 - i];
}
// invertido = { 5, 4, 3, 2, 1 }
✅
Dica: A classe Arrays oferece métodos prontos como Arrays.copyOf(v, n), Arrays.sort(v) e Arrays.fill(v, valor). Importe com import java.util.Arrays;.
06

Vetores multidimensionais (matrizes)

Uma matriz em Java é um vetor de vetores. O caso mais comum é a matriz bidimensional, usada para representar tabelas, grades e tabuleiros.

Java — matriz 2D
// Declaração e alocação: 3 linhas × 4 colunas
int[][] matriz = new int[3][4];

// Inicialização literal
int[][] tabela = {
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};

// Acesso: tabela[linha][coluna]
System.out.println(tabela[1][2]);  // 6 (linha 1, coluna 2)

// Percorrer com for duplo
for (int i = 0; i < tabela.length; i++) {
    for (int j = 0; j < tabela[i].length; j++) {
        System.out.print(tabela[i][j] + "\t");
    }
    System.out.println();
}

// Dimensões
System.out.println(tabela.length);     // 3 (linhas)
System.out.println(tabela[0].length);  // 3 (colunas da linha 0)

Erros comuns e boas práticas

  • ArrayIndexOutOfBoundsException — acessar índice menor que 0 ou maior/igual a length.
  • NullPointerException — declarar o vetor mas esquecer de criar com new.
  • Off-by-one — usar i <= v.length em vez de i < v.length.
  • Cópia por referência — usar int[] b = a acha que copia, mas ambos apontam para o mesmo vetor.
  • Modificar no for-each — o for-each itera sobre cópias; para modificar, use o for indexado.
Java — erros frequentes
int[] v = new int[5];

// ❌ Off-by-one
for (int i = 0; i <= v.length; i++)   // erro! <= dá AIOOBE
for (int i = 0; i <  v.length; i++)   // ✅ correto

// ❌ Vetor não inicializado
int[] arr;
arr[0] = 10;   // NullPointerException (esqueceu new)

// ❌ "Cópia" errada
int[] a = { 1, 2, 3 };
int[] b = a;     // b aponta para o MESMO vetor
b[0] = 99;      // a[0] também vira 99!
✅
Boas práticas: nomeie vetores no plural (notas, idades); use constante para tamanho se reutilizar (final int N = 10); sempre inicialize com new antes de usar.
